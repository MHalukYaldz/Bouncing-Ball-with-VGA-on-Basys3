# Bouncing-Ball-with-VGA-on-Basys3
FPGA-based VGA bouncing ball animation designed in VHDL for Basys3.

This project implements a simple bouncing ball animation on a VGA display using the Basys3 FPGA development board. The design is written in VHDL and includes VGA timing generation, pixel position control, and ball movement logic.

The project demonstrates basic video signal generation, coordinate-based graphics rendering, and real-time object movement on FPGA hardware.
